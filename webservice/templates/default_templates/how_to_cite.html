If you want to customize the text here, please

  • create a template file with the HTML content
  • store it in the folder for custom templates in the path templates/user_templates
  • in the config file, create a templates dictionary, if not yet there, and set the variable how_to_cite to point to you custom template: 
    templates:
  - how_to_cite: "custom_how_to_cite.html"

If instead you want to remove this section altogether, please set the how_to_cite variable as above to null: 

templates:
  - how_to_cite: null
